Created At : 2021-09-08 17:32
Views 👀 :

增：

​ 插入一行：

​ insert into 表名 [(字段列表)] values(值列表);

​ []表示可有可无

​ 注：自动增长的列不需要设置值,用null表示

​ 插入多行：

​ insert into 表名 [(字段列表)]

​ values(值列表1),(值列表2)….(值列表3);

​ 插入其他表已存在的数据：

​ insert into 目标表名[(目标表字段列表)]

​ select (源表字段列表) from 源表 where 条件表达式;

改：

​ update 表名 set 列1=值1,列2=值2,… [where 条件表达式];

删：

​ – 可以条件删除,标识列不重置,删除效率慢

​ delete from 表名 [where 条件表达式];

​ – 整表删除,标识列重置,效率快

​ truncate table 表名;

查：

​ select *|字段列表 from 表名

​ [where 条件表达式]

​ [group by 分组字段 [having 分组后条件]]

​ [order by 排序字段 asc(升)|desc(降)]

​ 1、字段列表可使用as取别名

​ 2、判断null：is null 和 is not null

​ 3、between … and … 在 … 之间

​ 4、in() 在… 之内

​ 5、like 模糊查询 通配符：% _ （没有[])

​ 6、not 取反 如：not in() 、 not like 、not between … and …

*group_concat()函数：*

​ 分组后，把其他列纵向拼接：

​ select stusex,group_concat(stuname)

​ from stuinfo GROUP BY stusex

​ – 如上：把同一性别的学生姓名拼接

​ 分页(查询部分行)：limit

​ select 列名 from 表 limit [start,] length;

​ start表示行索引(行索引从0开始),不填默认为0。length表示检索 多少行记录。

​ 注意：

​ 1.MySQL中没有TOP语句，而是使用limit分页

​ 2.limit语句应放在查询语句的最后，比order by还后

*去除重复数据：distinct*

​ select distinct 列名 from 表;

​ select count(distinct 列名) from 表

​ 注意：

​ 1.查询列中不能包含多个distinct语句，

​ 2.查询时最好只查询一个列，否则distinct可能会没有效果

两表连查：

​ select *|字段列表 from 表1 连接方式 表2 on 表1.公共列=表2.公共列

​ 连接方式：

内连接：inner join 两表都有的数据

​ 左外连接：left join 两表都有的数据及左表没有的数据

​ 右外连接：right join 两表都有的数据及右表没有的数据

​ 沒有全外连接（full join）

​ 交叉连接：cross join 会产生笛卡尔积（两张表的乘积）交叉连接不能使用on关键字

​

​ 简写：select *|字段列表 from 表1,表2 where 表1.公共列=表2.公共列 [and 其他条件表达式] – 相当于内连接

union：将两个查询结果合并为一个结果集

​ select 字段列表1 from 表1

Union

Select 字段列表2 from 表2

​

​ 注意：

​ 要求2个查询结构的字段列表个数和数据类型都必须一致

​ 合并结果时会去掉重复的数据

union all: 和union用法相同，但合并结果不会去掉重复的数据
