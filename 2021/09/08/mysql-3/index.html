函数

函数：分为系统函数和自定义函数

系统函数：系统提供，直接使用

聚合函数：

​ count() sum() avg() max() min()

日期函数：

​ now():当前日期时间，

​ 设置默认值时不能使用函数,需使用current_timestamp

​ curdate():当前日期

​ curtime():当前时间

获取日期或时间中某个单位的整数：

​ year(d):年份

month(d):月份

dayofmonth(d):日

hour(d):小时

minute(d):分钟

second(d):秒

weekday(d):一周中的第几天（周一为0）

week(d):一年中的第几周

dayofyear(d):一年中的第几天

日期运算：

datediff(d1,d2):计算d1-d2相差的天数

adddate(d,n):在d日期的基础上增加n天

​ subdate(d,n):在d日期的基础上减少n天

​ date_add(d,interval n type):在d日期时间的基础上增加/减少n单 位，type表示单位类型

例：

​ Select date_add(‘2016-01-01’,interval 1 day); – 2016-01-02

​ Select date_add(‘2016-01-01’,interval -1 day);– 2015-12-31

type单位：

格式转换：

​ date_format(d,f):按照f格式显示d日期

​ Select date_format(now(), ‘%Y%m%d’);– 20160103

​ Select date_format(now(), ‘%Y年%m月%d日’);– 2016年01月01日

 f格式：

数字函数：

​ round(x)：四舍五入后的整数

​ round(x,y)：四舍五入后的小数，y表示小数位数

​ truncate(x,y):保留y位小数的值，只舍不入

​ ceil(x)：返回大于等于x的最小整数(向上取整)

​ floor(x)：与ceil相反，返回小于等于x的最大整数（向下取整)

​ rand()：产生一个0~1的小数

字符串函数：

concat(s1,s2,…):将多个字符串拼接

concat_ws(s,s1,s2,….):将多个字符串使用s进行拼接

注意：在MySQL中不能使用+号来实现字符串拼接

char_length(s)：获取字符串字符长度

length(s):获取字符串字节长度

​ left(x,n):截取x字符串的前面n个字符

right(x,n):截取x字符串的后面n个字符

substring(x,start,length)：对x字符串进行截取，从start位置开始截取length个

Upper(s):转化为大写

Lower(s):转化为小写

replace(s1,s2,s3):在s1的字符串中查找s2字符串并替换成s3字符串

locate(substr,str) 查找substr在str中第一次出现的位置

locate(substr,str,pos) 查找substr在str中pos后第一次出现的位置

注：mysql中字符串下标索引从1开始

md5：

​ md5（s）：将字符串s转化为md5编码

特点：

​ md5编码:不可逆算法,无法反编译出原来的内容

​ 唯一算法：相同的内容生成相同的编码，不同的内容生成不同的编码

​ md5应用:

​ 存储密码

​ 验证文件一致性

类型转换函数：

​ convert(x,type): 将x转换为type类型并返回

​ type的取值：

字符类型：char(length)

日期类型：date、time，datetime

浮点型：decimal

整数：signed

条件函数：

if(条件表达式,值1,值2)

​ 符合条件表达式返回值1，否则返回值2

​ ifnull(值1,值2)

​ 值1为null,返回值2，否则返回值1

自定义函数：

​ 定义：

​ create function fn_test(id int)returns int

Begin

​ set id=id+10;

​ return id;

end;

定义函数注意事项：

必须有返回类型

函数名使用fn_开头命名

函数中不能产生查询结果集

调用：

select fn_test(5); –结果为1

注意：

可以在增删改查语句中使用函数，参数可以是列名

删除函数：

drop function 函数名
