测试题

1.将一个保单记录（保单号：001；被保人：张三，产品类型： 109；保险起期2017-01-01；保险止期：2017-12-31）插入t_policy_base表中。

insert into t_policy_base values(001,”张三”，109，“2017-01-01，“2017-12-31”)

2.删除车辆表中车辆中保单编号为001的数据。

delete from t_vehicle where vo=001

3.删除车辆表中保单号为001的数据。

delete from t_policy_vehicle where po=001

4.将保单号为002的被保险人改为“李四”。

updete t_policy_base set iname=”李四” where po=002

5.查询保单表中起保时间大于2017-05-01的所有数据，按照保单号降序排列。

select po from t_policy_base order by sdate >2017-05-01 Desc

6.查询产品类型为109的所有车辆编号，保单号，被保险人

select vo,po,iname from t_policy_base where (select *from t_policy_base as tb inner join t_policy_vehicle as tv on tb.po=tv .po and pc=109)
